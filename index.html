Tentang Saya

S-1 Sistem Informasi

Universitas Negeri Semarang


Saya seorang lulusan dari prodi Sistem Informasi dengan antusias tinggi dalam perkembangan teknologi informasi. Saya memiliki pengalaman magang dan project sebagai Web Developer dengan membangun sebuah aplikasi berbasis web. Familiar menggunakan Bahasa pemrograman Javascript dan PHP serta penggunaan Bootstrap dan Tailwind sebagai CSS. Tertarik pada bidang kerja pada pengembangan teknologi Informasi terkhusus pada Web Developer, Bussiness Development, dan juga bidang kerja lainnnya yang relevan.


Saya adalah seorang yang mampu berpikir analisis dalam suatu permasalahan, kritis terhadap pekerjaan,dan mempunyai ide inovatif. Seringkali saya juga memiliki minat dalam menganalisis suatu tren perkembangan teknologi, sosial media, dan ide bisnis inovatif. Senang bekerja dalam tim, mampu berkomunikasi dengan baik, dan teratur dalam mengerjakan berbagai hal menjadi sikap positif bagi saya. Menyelesaikan tugas dan tanggung jawab adalah moto saya dalam melakukan berbagai pekerjaan.
